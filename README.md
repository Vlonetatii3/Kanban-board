## Modern Kanban-style task management application.
Built with Spring Boot and a RESTful API architecture.
It allows users to organize tasks by status (e.g. To Do, In Progress, Done, Cancelled) and manage them through a clean and intuitive interface.

## 🚀 Features

- Create, update, and delete tasks

- Organize tasks by status (Kanban board style)

- Real-time board overview with task counters

- Clean and modern UI

- RESTful API backend

- Persistent storage with PostgreSQL

- Data access using JPA/Hibernate

## 🛠️ Tech Stack

**Backend:** Java, Spring Boot

**Database:** PostgreSQL

**ORM:** JPA / Hibernate

**API:** REST

**Frontend:**  HTML, CSS, JS
## 📦 Project Structure

- Layered architecture (Controller, Service, Repository)

- Entity-based data model

- DTOs and clean separation of concerns

- Scalable and maintainable backend design

## ▶️ How to Run

1. Clone the repository

2. Configure your PostgreSQL database in ``application.properties`` or ``application.yml``

3. Run the Spring Boot application

4. The API will be available at: ``http://localhost:8080``

## 🎯 Purpose

This project was built to practice and demonstrate skills in Java backend development, REST API design, and database-driven applications using modern Spring Boot best practices.
